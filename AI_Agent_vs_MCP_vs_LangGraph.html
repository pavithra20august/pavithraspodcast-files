An AI Agent is a smart digital assistant that can:

What It Is:

MCP is like a recipe or blueprint that tells your AI system:

  • What steps to follow
  • What data to pass between steps
  • What tools or models to call
  • How to update or maintain memory (context)

üîó What Is LangGraph?

üíª Imagine This:

You're a developer who wants to build the workflow using MCP principles.

What It Is:

LangGraph is a Python framework to build stateful, multi-step AI workflows, inspired by the MCP pattern.

It lets you connect steps (nodes) in a graph. Each node can:

  • Use an LLM (OpenAI or Gemini)
  • Call tools (search, DB, embedding)
  • Modify shared memory (state)
